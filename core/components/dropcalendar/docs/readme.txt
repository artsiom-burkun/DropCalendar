A JavaScript event calendar (https://fullcalendar.io/)
with ajax event management for MODx Revolution.

Javascript libraries:
 - jQuery v3.3.1
 - jQuery UI 1.12.1
 - Bulma v0.7.4
 - Fullcalendar v3.9.0
 - Moment 2.18.1

--------------------
 Usage:
 [[DcAdmin]]
 is equal
 [[DcAdmin?
    &tpl=`DcAdmin`
    &modal=`DcAdminModal`
    &locale=`ru`
    &calendar=``
    &useBulma=`1`
    &useFullcalendar=`1`
 	&useJquery=`1`
 ]]
 by default.

 Change &calendar value to work with different calendars.
 Change &locale value to set your locale.
